## 🔄 **Workflow**

### Complete Process
1. 📤 **Upload** manga images (5-50 panels)
2. 🌐 **Select** language (Indonesian or English)
3. 🤖 **AI analyzes** images and story
4. 🎙️ **Generates** narration text
5. 🎬 **Creates** 9:16 video with effects
6. ✅ **Preview** and download result

### Time Breakdown
- Upload: 5-10 seconds
- Analysis: 10-30 seconds
- Video generation: 20-60 seconds
- Total: 1-2 minutes average

## 🎓 **How It Works**

### Technology Stack
- **Frontend**: Next.js 14, React, TypeScript
- **Styling**: TailwindCSS
- **Video**: Canvas API, MediaRecorder
- **AI**: Hugging Face, OpenAI (optional)
- **Storage**: Local file system

### Architecture
```
User uploads images
       ↓
Server receives files
       ↓
AI analyzes panels
       ↓
Generates narration (ID/EN)
       ↓
Creates video manifest
       ↓
Client renders with effects
       ↓
User downloads video
```
